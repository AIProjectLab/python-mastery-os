> A complete interactive Python course, a real code editor, and an AI tutor — **in one free HTML file**. No install. No account. No setup.

**▶ [Try it live in your browser](https://aiprojectlab.github.io/python-mastery-os/)** · **[Download the file](https://github.com/AIProjectLab/python-mastery-os/releases/latest)** (works offline after first load)

## What's inside

- **30 story-based lessons** across 6 stages — from your first variable to building a real app with saved data. Every lesson ends with a **⭐ YOUR TURN** practice task.
- **Real Python in your browser** — code runs via Pyodide, right next to the lesson. `input()` works, files work, everything works.
- **Split view**: annotated story code on the left, your free practice editor on the right.
- **AI tutor built in** — select any code and ask *Explain / What's wrong? / Improve / Custom*. Full code reviews with presets. Automatic error explanations.
- **4 AI providers**: Google Gemini (free tier), OpenAI, Anthropic Claude — or **LM Studio for 100% local, private AI** on your own computer.
- **Progress that sticks**: completion tracking, daily streak, per-lesson notes, course index — all saved in your browser.

## How to use

1. Open the file (or the live link) in Chrome/Edge/Firefox.
2. Start with Story 01. Read → RUN → edit → do the ⭐ YOUR TURN task.
3. Optional: click **✦ AI Settings** and add a free Gemini key ([get one here](https://aistudio.google.com/app/apikey)) to unlock the AI tutor.
4. Come back tomorrow. Keep the streak alive. 🔥
